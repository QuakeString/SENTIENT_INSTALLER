# SENTIENT Installer

A graphical installation helper for the SENTIENT platform. On a fresh machine it
checks every prerequisite, provisions what's missing (WSL2 + Docker Engine on
Windows), deploys the SENTIENT stack from the public registry, and installs the
backup app — all from a single wizard.

**v1 target:** Windows, Docker Engine inside WSL2 (no Docker Desktop), online
image pull. See [docs/RESEARCH_AND_PLAN.md](docs/RESEARCH_AND_PLAN.md).

Built with Rust + Tauri (same stack as the SENTIENT Backup & Restore app).
